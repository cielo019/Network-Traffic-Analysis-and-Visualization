# Network-Traffic-Analysis-and-Visualization
To monitor, analyze, and generate reports on network traffic to understand packet flow, bandwidth usage, and detect potential network threats.

Tools Used:
  - Wireshark (packet capture)
  - Python (automation & analysis)
  - Libraries: pyshark, pandas, matplotlib, seaborn
 

Expected Outcome:
  - Understand packet-level communication
  - Visualize traffic patterns
  - Identify suspicious or abnormal traffic behavior

Domain: Networking | Cybersecurity | Data Analysis
